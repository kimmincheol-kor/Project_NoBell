Owner App

[사장 (Office)]
1. 사장 회원정보 관리
- 현재 정보 출력 & 수정 기능

2. 업체 정보 수정
- 현재 정보 출력 & 수정 기능

3. 이벤트 관리
- 이벤트 리스트 출력
- 이벤트에 대한 CRUD

4. 리뷰 확인 & 답글
- 리뷰 리스트 출력
- 리뷰에 대한 답글 CRUD

5. 통계 데이터 확인
- 일별, 월별, 년도별 데이터 열람 (수익, 평균 손님)


[홀 (Field)]
1. 주문 처리
- { 처리되지 않은, 처리 중인, 처리 된 } 3가지로 분류. (시간, 테이블 정보)
- 처리되지 않은 주문 -> 접수 -> 처리 중인 주문 -> 완료 -> 처리 된 주문

2. 테이블 관리
- 현재 테이블 현황 표시
- 각 테이블별 주문상태, 방문자 정보 확인
- 방문신청 목록 확인 -> 테이블 배정

3. 예약 신청 처리
- 예약 신청 목록 확인
- 방문자 수, 예약 시간 확인 -> 승인, 거절 처리

4. 계산 처리
- 계산 요청 목록 확인 -> 계산 처리 (직접결제시 테이블 방문, 자동결제시 바로 처리)
